CSS

CSS is of three types
  1. Inline CSS - where each item is styled individually, in the tag definition
    • This can lead to messy code and needs to be avoided
    • NOTE : Inline CSS gets priority over Internal CSS
  2. Internal CSS - where CSS is defined in the HEAD tag
    • This will increase the size of the HTML ( header section)
  3. External CSS - CSS is defined in an external .css file and included
    • Neat soluton if there is a lot of styling
    • Style sheet can be resued across several .HTML files
