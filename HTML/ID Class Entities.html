ID and Class

ID is given to a DIV ( or any HTML tag for that matter) to identify it within a JS or CSS code
CLASS is a grouping of individual ID's that can be given a set of common properties via CSS class definition A tag can have multiple CLASS definition ( space delimited ) and it will inherit properties from all of them
Shotcuts
# is for id
if we type [div#mainID] it will expend to ["]div id="mainID" ... ]

. is for CLASS
if we type [div.border1] it will expend to ["]div class="border1" ... ]
